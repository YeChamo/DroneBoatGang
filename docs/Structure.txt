1. Repo + Version Control Setup
Deliverables: GitHub repo with .gitignore, README.md, and 
branch workflow. All members can clone, commit, and push.

2. Toolchain Setup
Deliverables: STM32CubeIDE installed and verified with a simple build, React Native + TypeScript 
environment running a test app, and /docs/setup.md with install steps.

3. Requirements & Use-Case Definition
Deliverables: /docs/requirements.md containing functional and non-functional requirements, plus
a few user stories with acceptance criteria.

4. System Architecture & Interface
Deliverables: /docs/architecture.md with a block diagram, list of modules (firmware + mobile app),
and description of key interfaces (BLE, LoRa, HAL APIs).
